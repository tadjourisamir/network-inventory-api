📌 Project Overview

This project is a RESTful API built with Flask to manage network equipment like switches, routers, and access points.

It includes a web frontend and full CRUD functionality for local development and testing.

🛠️ Technologies Used

  • 🐍 Python 3.12.3
  • 🫙 Flask 3.1.0
  • 🛢 SQLite 3.45.3
  • 🌐 HTML & CSS
  • 🐳 Docker 28.0.4

📡 API Endpoints

The API provides a set of routes to interact with your network inventory data. All data is sent and received as JSON.

  • GET /equipements — Fetch all equipment
  • GET /equipements/<id> — Get a specific equipment by its ID
  • POST /equipements — Add new equipment (expects JSON)
  • PUT /equipements/<id> — Update an existing equipment entry
  • DELETE /equipements/<id> — Delete equipment (⚠ irreversible)
  • GET /export?format=csv — Export inventory as CSV
