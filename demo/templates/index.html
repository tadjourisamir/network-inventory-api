📌 Project Overview

This project is a simple RESTful API built with Flask to manage network equipment like switches, routers, and access points.

It's part of a personal learning path to create structured, documented APIs using modern tools and best practices.

🛠️ Technologies Used

  • 🐍 Python 3.12.3
  • 🫙 Flask 3.1.0
  • 🛢 SQLite 3.45.3
  • 🌐 HTML & CSS
  • ⚙️ JavaScript

📡 API Endpoints

The API provides a set of routes to interact with your network inventory data. All data is sent and received as JSON.

  • GET /equipements — Fetch all equipment
  • GET /equipements/<id> — Get a specific equipment by its ID
  • POST /equipements — Add new equipment (expects JSON)
  • PUT /equipements/<id> — Update an existing equipment entry
  • DELETE /equipements/<id> — Delete equipment (⚠ irreversible)

ℹ️ This online interface is a demo version: only GET requests are available.

The full version (with add, update, and delete features) is available on GitHub.

🚀 How to Use the API

Use any HTTP client like curl, Postman, or your frontend app to interact with the endpoints.

All responses are returned in application/json format. For detailed usage and examples, check the README.
